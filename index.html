Respostas

  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

    O atributo input placeholderespecifica uma dica curta que descreve o valor esperado de um campo de entrada.

  2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    Serve para o input checkbox ou radio já venha selecionado\ marcado sem necessidade do usuario marcar.

  3. Na tag <input> para que serve o atributo "value"?

    O atributo value é um valor pre determinado que não desaparece após interagir com o input.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    O atributo name

  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    Serve para adicionar uma descrição ao botão, o for serve para atribuir associar o label a um elemento do formulario

  6. Na tag <option> para que serve o atributo "selected"

    Serve para definir como padrão a opção selecionada no dropdown

  7. Nas tags de formulário para que serve o atributo "required"?

    Especifica que um campo de entrada deve ser preenchido antes de enviar o formulário,que para finalizar o campo precisa ser preenchido para prosseguir

  8. Para que serve a tag <textarea>

    Serve para abrir um campo de texto,como se fosse um campo de observações
